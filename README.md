Разрабатываю веб-системы, Telegram-ботов, mini apps, CRM и автоматизации для бизнеса.

Делаю прикладные решения для бизнеса:

- Telegram-боты: заявки, уведомления, оплаты, рассылки, интеграции с CRM.
- CRM / mini-CRM / POS-системы: заказы, клиенты, смены, услуги, зарплаты, аналитика.
- Django/FastAPI-сервисы: REST API, админ-панели, личные кабинеты, интеграции.
- AI-боты и AI-интеграции: обработка данных, генерация контента, ассистенты и API.
- Внутренние сервисы и автоматизация рутины.

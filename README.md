Платформа для генерации фото, видео, музыки и общения с ИИ-чатом.

Реальный коммерческий продукт в продакшене. В репозитории — безопасная публичная версия: интерфейс, архитектура, скриншоты и часть структуры проекта без приватной логики.

### [Pegas Auto CRM](https://github.com/qXstay/pegas-auto-crm)

CRM/POS-система для сети шиномонтажей: заказы, клиенты, автомобили, запись, смены, зарплаты и аналитика.

Реальный коммерческий проект. В репозитории — скриншоты, описание архитектуры и обезличенные примеры кода.

**Стек:** Next.js, TypeScript, Prisma, PostgreSQL, роли, права доступа, учёт филиалов, финансовая история заказов.

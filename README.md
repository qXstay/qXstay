### [NeuroFox AI](https://github.com/qXstay/neurofox-ai)

**Что это:** AI-платформа для генерации фото, видео, музыки и общения с AI-чатом.

**Что внутри:** пользователи, токены, тарифы, платежи, история генераций и интеграции с AI-сервисами.

### [NeuroFox AI Telegram Bot](https://github.com/qXstay/neurofox-ai-telegram-bot)

**Что это:** коммерческий Telegram-бот для генерации фото, видео, музыки и работы с AI-чатом.

**Что внутри:** платежи, баланс токенов, тарифы, история генераций и реферальная программа.

**Статус:** работает в продакшене, принимает платежи, ведёт баланс токенов и помогает проекту получать пользователей.

**Telegram:** [@Malinovsky_AI_bot](https://t.me/Malinovsky_AI_bot)

### [Pegas Auto CRM](https://github.com/qXstay/pegas-auto-crm)

**Что это:** CRM/POS-система для сети шиномонтажей.

**Что внутри:** заказы, клиенты, автомобили, запись, смены, склад, зарплаты и аналитика по филиалам.

**Результат:** система работает в продакшене для 3 филиалов и помогает владельцу контролировать заявки, сотрудников, склад и расчёты.
